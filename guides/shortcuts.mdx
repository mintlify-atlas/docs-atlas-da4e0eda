Steam Controller maps button combinations to system actions across every controller mode it provides. Depending on which profile is active — Desktop, X360 (with Rumble), DS4 (with Rumble), Steam, or Steam With Shortcuts — the same button presses may behave differently or do nothing at all.

<Note>
  These shortcuts only work when **Steam Controller** is running. If the application is not active, hardware buttons pass through to Windows or Steam unmodified.
</Note>

## Switching modes

You switch between controller profiles using the **Options** button (the three-dot menu button on the right side of the Steam Deck):

- **Hold Options for 1 second** — cycle to the next profile in the list.
- **Hold Options for 2 seconds** — toggle desktop mode on or off. This shortcut is not available in the Steam or Steam With Shortcuts profiles.

<Tip>
  **STEAM + DPad Up** sends `CTRL+ALT+U`, which is the default hotkey for toggling fullscreen scaling in apps like [Lossless Scaling](https://store.steampowered.com/app/993090/Lossless_Scaling/). If you use fullscreen upscaling, this shortcut lets you toggle it without leaving the controller.
</Tip>
