Steam Deck Tools is a collection of four Windows applications built to make running Windows on your Steam Deck feel as polished and capable as SteamOS. From a reimplemented SteamOS fan curve to full Xbox 360 controller emulation, these tools give you fine-grained control over your device's hardware directly from the Windows desktop.

<Warning>
  This software is provided on a best-effort basis and involves direct manipulation of kernel memory. It **can break your Steam Deck**. Read the [Risks](/reference/risks) page before use.
</Warning>

## Key capabilities

- **SteamOS fan curve** — Fan Control reimplements the `jupiter-fan-control.py` algorithm from SteamOS for quieter, smarter cooling
- **On-screen performance stats** — Six overlay modes from a minimal FPS counter to a full hardware dashboard
- **TDP control (3W–15W)** — Set slow and fast TDP limits independently to balance performance and battery life
- **Controller emulation** — Desktop, X360 (with haptics), DS4 (with gyro via DS4), and Steam modes
- **Battery charge limit** — Cap charging at 70%, 80%, 90%, or 100% to protect battery longevity (LCD and OLED)
- **SMT toggle** — Enable or disable AMD CPU hyperthreading per-session
- **Auto-updates** — Built-in updater checks for new releases on startup
